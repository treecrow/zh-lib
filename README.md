# @treecrow/tool-lib

> 个人使用的 js 工具库

## 工具库支持的特性

- 可分块引入，也可整体引入
- 可分块打包成压缩文件，直接在 html 文件中使用
- 库开发者美滋滋的写 ES6+的代码；
- 库使用者能够运行在浏览器（ie6-11）和 node（0.12-10）中
- 库使用者能够使用 AMD 或 CMD 模块方案
- 库使用者能够使用 webpack、rollup 或 fis 等预编译工具
